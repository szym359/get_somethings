# Codex Skill 学习资料

这个目录是项目内的 skill 学习资料库，不是实际会被 Codex 自动发现的 skill 目录。

实际可触发的项目级 skill 应该放在：

```text
.agents/skills/你的-skill-name/SKILL.md
```

本目录用于学习、查阅和复制模板：

```text
ai/skills/
  README.md
  01-skill-是什么.md
  02-skill-目录结构.md
  03-skill-触发方式.md
  04-agents-md-和-skill-区别.md
  05-英文文档中文学习版.md
  examples/
    minimal-skill-template.md
    practical-skill-template.md
```

建议学习顺序：

1. 先读 `01-skill-是什么.md`
2. 再读 `02-skill-目录结构.md`
3. 然后看 `examples/minimal-skill-template.md`
4. 最后读 `04-agents-md-和-skill-区别.md`

一句话理解：

`AGENTS.md` 是项目长期默认规则，skill 是某个任务场景才调用的专用说明书。
